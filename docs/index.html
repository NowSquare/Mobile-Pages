Requirements

You need a domain or subdomain. The script does not work in a subdirectory (e.g. localhost/mobile-pages or example.com/mobile-pages).

  • Apache web server
  • PHP >= 7.1.3
    • SQLite PHP Extension
    • BCMath PHP Extension
    • Ctype PHP Extension
    • JSON PHP Extension
    • Mbstring PHP Extension
    • OpenSSL PHP Extension
    • PDO PHP Extension
    • Tokenizer PHP Extension
    • XML PHP Extension
Install

Upload all files in the public directory to your webroot. Open the url where you uploaded the script to in your browser and you will see an installation form.

Update

In the latest Zip file you download there is an updates directory:

\---productinnovation-mobile-pages-[commit_hash]
    +---docs
    +---public
    \---updates
            incremental-update-x.x.x.zip

The file incremental-update-x.x.x.zip contains all updated files. You do not have to worry about your current version, you can simply overwrite the files of your installation with the files in the update Zip.
